# Plasma Video Wallpaper
![Screenshot](http://i.imgur.com/mFf8NZp.jpg)
Build instructions
----------
```bash
cmake .
make
sudo make install
```
Restart plasma to load the wallpaper
(in a terminal or in krunner type:
`kquitapp plasmashell`
and then
`plasmashell`)
then go to wallpaper settings and select it
